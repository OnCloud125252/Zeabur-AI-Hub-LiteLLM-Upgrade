## 檔案摘要

### 升級變更日誌 (upgrade-changelog-v1.79-to-v1.81.md)

v1.79.0 與 v1.81.12-stable.1 之間所有變更的完整分析：

| 類別 | 數量 | 重要項目 |
|------|------|----------|
| 重大變更 | 7 項 | Docker 映像檔遷移、Python 版本、OpenAI SDK v2 |
| 新功能 | 50+ | A2A 代理程式、政策引擎、存取群組、技能 API |
| 錯誤修復 | 100+ | 記憶體洩漏、工具呼叫、串流、成本追蹤 |
| 效能改進 | 15+ | 延遲降低 21%、LRU 快取 |
| 安全性修復 | 10+ | 金鑰洩漏防護、SSRF 防護、CVE 修補 |

### 資料庫結構遷移 (db-schema-migration-v1.79-to-v1.81.md)

包含手動 SQL 遷移腳本的完整結構變更分析：

| 階段 | 版本範圍 | 新增資料表 | 修改資料表 |
|------|----------|------------|------------|
| 階段 A | v1.79.0 → v1.80.11 | 8 | 5 |
| 階段 B | v1.80.11 → v1.81.12 | 7 | 7 |

**安全替代方案**：適用於生產環境的 `prisma db push` 替代方法，確保資料安全。

### PR #16895：Gemini Thought Signatures（初始）

Gemini thought signature 問題的初始修復分析：

- **合併日期**：2025-11-21
- **修復方法**：將簽名嵌入工具呼叫 ID 格式 `call_<uuid>__thought__<signature>`
- **變更檔案**：Factory、Vertex AI 處理器、測試
- **程式碼變更**：+374 行

### PR #18374：Gemini Thought Signatures（最終）

將功能從 beta 提升至穩定版的改進：

- **合併日期**：2025-12-23
- **改進項目**：前置呼叫鉤子（Pre-call Hook）、OpenAI Agents SDK 相容性
- **變更檔案**：Utils（前置呼叫鉤子）、測試
- **程式碼變更**：+449 行、-186 行

### PR 相容性矩陣

追蹤哪些 LiteLLM 版本包含關鍵修復：

| 版本 | PR #16895 | PR #18374 | 狀態 |

| v1.80.11-stable | ✅ | ✅ | **最低需求版本** |
